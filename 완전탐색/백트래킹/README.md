## 백트래킹
  
- 탐색하면서 특정 조건을 만족하지 않으면 해당 가지(branch)를 더 이상 탐색하지 않고 이전 단계로 되돌아가는(backtrack) 방식으로 동작하는 알고리즘
- 특정 조건을 통해 유망하지 않는 노드라면 해당 탐색은 더 이상 이뤄지지 않게함 
- 주로 순열, 조합, 부분집합 등의 조합적 문제를 해결하는 데 사용됨
- 일반적으로 재귀로 구현됨
  
  
```
           N = 3 , M = 3   
       /        |         \
      1         2          3
    /  \       /  \       /  \
   2    3     1    3     1    2
   |    |     |    |     |    |
   3    2     3    1     2    1
```
